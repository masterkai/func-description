GPS 衛星導航系統，強大而精準

Ambit 3 Vertical 腕錶能夠接收 GPS 衛星訊號、導航準確度高、 路徑修正迅速。能迅速計算出您目前的速度與位置。

在【Movescount 運動平台】規劃路線
  • 【Movescount 運動平台】上，有超過 300 萬條公開路線。
  • 在地圖上創建一條【路線】，讓您輕鬆自在地使用 Suunto 腕錶從事健行、騎乘自行車、跑步 及登山越野等活動，足跡踏遍叢林野外，體驗冒險精神
  • 在規劃路線時，隨時檢視、調整路線的高度分佈，並分享您的公開路線。
  • 以 .kml 及 .gpx 格式匯入/匯出路線。
豐富的地圖資訊，協助您規劃路線
  • 【Movescount 運動平台】上的 Mapbox、Google 、德高等豐富的圖資，支援地形圖、等高線地圖、衛星圖、黑圖、及光圖，提供您豐富的地理資訊。
  • 即使未知地形中的越野小徑也不例外，都能協助路線的詳細規劃。
  • 可預先確認地表和地形的詳細信息，方便選擇搭建帳篷的最佳區域。
【路線】導航
  • 在【Movescount 運動平台】上，建立一條路線，利用GPS導航系統，沿著腕錶的指示帶領您至目的地。
  • 在【Movescount 運動平台】上，規劃路線，設定【興趣點】，【興趣點】可以是賽事補給點、出遊地標、登山紮營地點、水 源地點等等。在【路線導航】過程中，隨時掌握【興趣點】的相 對位置。
【興趣點】導航
  • 您可以將【目前的位置】，或在【Movescount 運動平台】上，事先將重要的地點，設定成【興趣點】
  • 當導航至興趣點，在您靜止不動時，三角形顯示你的興趣點方位。 請將您的三角形置於兩條實線當中，即可確保到達您的目的地。
  • 接近【興趣點】時，腕錶會以警示音，提醒您正在接近中。
【路線高度剖面圖】導航
  • 導航時，Ambit 3 Vertical 會以【高度剖面圖】的方式呈現,您可以隨時掌握您目前位置的高度、上升、下降、坡度、及路線上剩餘的峰頂數等訊息。
3D 距離

在預設條件下，Ambit 3 Vertical 計算距離時會同時考慮上升和下降的高度變化量。當您在有些起伏的地形或是高山上運動時，這種方式可提供更為真實的距離量測結果。在這種地形下，3D 距離會比 2D 距離還要長一些。

爬升歷程

Ambit 3 Vertical 可根據記錄的步數，監測您全程的爬升歷程。爬升歷程可以記錄每天或甚至是整個使用期間所累積的高度，幫助您瞭解整個過程中的高度變化。

支援 STRYD 跑步專用功率計
全世界第一個跑者用功率計

跑步時，跑步功率計讓您踏出的每一步，都被精準的計算，藉由功率數據的輸出，校正您的跑步姿勢與配速，進而提升您的跑步效率與成績。除了跑步功率之外，Stryd 跑步功率計還可以偵測步頻、觸地時間、垂直振幅、腿部彈性、配速、及距離。

FusedSpeed™

獨特地結合 GPS 和手腕加速感應器讀數，可更精準測量您的跑步速度。GPS 訊號會根據手腕加速狀況自行調整篩選，提供更精準規律穩定的跑步速度讀數，並對速度變化做出更快的反應。

您進行的訓練需要高度反應速度讀數時（例如在不平坦地面跑步或間歇訓練期間），FusedSpeed™ 對您的幫助最大。
進行的運動訓練區域暫時失去 GPS 訊號（例如路跑進入隧道時），在 GPS 被校準的加速計的協助下，Suunto Ambit 3 依舊能夠繼續顯示精準的速度讀數。

跑步效率

跑步成績等級也會加入計算跑步效率，這是跑步技巧的測量方式。此測量方式也會加入計算額外因素，例如地形、跑步路面、天氣及跑步配備。

跑步期間

您的 Suunto Ambit 3 會在您跑步時提供即時跑步表現回饋。即時回饋會以圖形顯示在 Running 預設的運動模式中。您可以將此圖形新增到任何使用跑步活動類型的運動模式。即時差異（請參閱下方）也可以在自訂跑步運動模式中作為資料欄位使用。在您跑步時，圖形顯示畫面會顯示您的四個資料點，如下圖所示。

  1. 基線：
    每次跑步時，都會校正您的跑步表現基線，以補償外在變化（例如地形）與等待您的心跳頻率穩定。
  2. 即時等級：
    這是針對您目前每公里/英哩跑步狀況顯示的即時跑步表現等級。
  3. 即時差異：
    顯示已校正之跑步表現基線與您目前跑步之即時跑步表現等級間的差異。
  4. 距離：
    目前記錄的總距離。圖形顯示最後六公里。

如果您是跑步新手或已經有一段時間沒有跑步，您的跑步表現一開始可能會很差。但是隨著您的身體狀況與跑步技術改善，您應該會看到跑步表現有相對應提升。在 4-20 週之後，您的跑步表現等級最多可能會提高 20%。如果您的跑步表現等級已經很好就很難再進一步提升。在這樣的情況下，最好將跑步效能等級當作耐力指標。即時差異指標後面會提供與每日跑步表現及跑步時產生疲勞有關的詳細資訊。耐力跑者可使用此資訊來瞭解在長跑時可以管理多少疲勞量。在比賽時，此資訊可協助您正確配速。下面的圖示說明是三個跑步時的跑步表現等級範例。

跑步成績

跑步成績等級是由 Firstbeat 提供的一種綜合測量方式，可測量您的身體健康狀況及跑步效率。

體適能

跑步成績等級是使用您的 VO2max 估計值，而 VO2max 是有氧健身與耐力表現的全球標準。VO2max 代表您身體在輸送及利用氧氣方面的最大能力。您的心臟、肺部、循環系統，以及肌肉產生能量時的氧氣利用能力等因素都會影響 VO2max。VO2max 是評估運動員耐力表現時最重要的單-標準。此外，研究指出 VO2max 也是衡量健康狀況的重要指標。在達到最佳跑步效率時，您的跑步成績等級會對應於您的真實 VO2max（ml/kg/min）
